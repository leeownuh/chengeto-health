# CHENGETO Brand Assets

Place the official project logos in this folder so the UI, favicon, and PWA icons use them.

Expected filenames:

- `chengeto-logo-rect.png` (rectangle / wordmark)  
- `chengeto-logo-square.png` (square / icon)

Notes:

- If these files are missing, the UI falls back to a simple "C" mark (so the app still runs).
- Once the files exist, refresh the browser (and for PWA installs, reinstall/clear site data) to see updated icons.
